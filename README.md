# sysgraph
A utility that converts a system specification XML file to a graph.

## Build Requirements
* [The Rust and cargo toolchain](https://www.rust-lang.org/en-US/)

## Building
```bash
$ cargo build
```

## Running
```bash
$ cargo run -- --help
```
or
```bash
$ sysgraph --help
```
